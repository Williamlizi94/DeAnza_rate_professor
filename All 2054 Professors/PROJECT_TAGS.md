# De Anza College 教授评分系统 - 项目标签集

## 技术栈标签

### 后端技术
- `Python` - 主要编程语言
- `FastAPI` - 现代Web框架，用于构建RESTful API
- `Uvicorn` - ASGI服务器，用于运行FastAPI应用
- `Requests` - HTTP库，用于网络请求和数据抓取

### 前端技术
- `HTML5` - 网页结构
- `CSS3` - 样式设计（包含渐变背景、动画效果）
- `JavaScript` - 前端交互逻辑
- `Bootstrap 5.3` - UI框架，响应式设计
- `Font Awesome 6.4` - 图标库

### 数据格式
- `JSON` - 数据存储和API响应格式
- `CSV` - 数据导出格式

## API端点标签

- `GET /` - 首页
- `GET /professors` - 获取所有教授
- `GET /professors/name/{name}` - 按姓名搜索
- `GET /professors/department/{department}` - 按部门筛选
- `GET /search` - 搜索教授
- `GET /stats` - 统计信息
- `GET /departments` - 部门列表
- `GET /docs` - API文档
