# De Anza College 教授评分系统 - Web界面

## 功能特点

✨ **现代化的Web界面**，包含以下功能：

### 🎨 界面特色
- **渐变背景设计** - 美观的紫色渐变背景
- **卡片式布局** - 清晰的教授信息卡片展示
- **响应式设计** - 完美适配桌面和移动设备
- **悬停动画效果** - 鼠标悬停时卡片会提升显示

### 🔍 搜索功能
- **按姓名搜索** - 支持部分匹配搜索
- **按部门筛选** - 下拉菜单选择部门
- **评分筛选** - 设置最低评分要求
- **难度筛选** - 设置最高难度限制
- **组合筛选** - 支持多个条件组合搜索

### 📊 统计信息
- 总教授数
- 平均评分
- 总评价数
- 部门数量

### 📝 教授信息展示
- 姓名和部门
- 平均评分（星级显示）
- 难度评分
- 评价数量
- 愿意再次选课百分比
- 最新评价（最多显示3条）

### 📄 分页功能
- 支持大量数据的分页浏览
- 每页显示20条记录
- 页面导航按钮

## 访问方式

1. **启动API服务器**：
   ```bash
   python api.py
   ```

2. **在浏览器中访问**：
   ```
   http://localhost:8000
   ```

3. **API文档**：
   ```
   http://localhost:8000/docs
   ```

## 使用说明

### 搜索教授
1. 在搜索框中输入教授姓名（支持部分匹配）
2. 或选择部门进行筛选
3. 可设置最低评分和最大难度
4. 点击"搜索"按钮或按Enter键

### 查看详细信息
- 每个教授卡片显示基本信息
- 点击展开查看最新评价
- 评分使用星级显示

### 浏览数据
- 使用底部分页按钮浏览更多结果
- 每页显示20条记录

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **UI框架**: Bootstrap 5.3
- **图标**: Font Awesome 6.4
- **后端**: FastAPI
- **API**: RESTful API

## 文件结构

```
.
├── api.py                          # FastAPI后端服务器
├── static/
│   └── index.html                  # Web前端界面
├── rmp_deanza_all_professors.json  # 教授数据文件
└── README_WEB.md                   # 本文档
```

## 注意事项

- 确保 `rmp_deanza_all_professors.json` 文件存在于项目根目录
- 服务器默认运行在 `http://localhost:8000`
- 如需修改端口，请编辑 `api.py` 文件末尾的端口配置
